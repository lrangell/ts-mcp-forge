## Quick Example

```typescript
import { Tool, MCPServer } from 'ts-mcp-forge';

class CalculatorServer extends MCPServer {
  @Tool()
  add(a: number, b: number): number {
    return a + b;
  }

  @Tool()
  subtract(a: number, b: number): number {
    return a - b;
  }
}

const server = new CalculatorServer();
await server.connect(process.stdin, process.stdout);
```

## Why TS MCP Forge?

- **Simple**: Define your MCP server with decorators and let the framework handle the protocol
- **Type-Safe**: Full TypeScript support with automatic type inference
- **Flexible**: Support for tools, resources, prompts, and dynamic registration
- **Well-Tested**: Comprehensive test coverage including MCP compliance tests
- **Production-Ready**: Used in production MCP servers
